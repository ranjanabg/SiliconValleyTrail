# Silicon Valley Trail

A CLI-based game built with Java and Maven.

## Prerequisites

- Java 17+
- Maven 3.x+

## Commands

| Action | Command |
|---|---|
| Compile | `mvn compile` |
| Run | `mvn compile exec:java` |
| Build JAR | `mvn package` |
| Run JAR | `java -jar target/silicon-valley-trail-1.0-SNAPSHOT.jar` |
| Clean build artifacts | `mvn clean` |
| Clean and recompile | `mvn clean compile` |
